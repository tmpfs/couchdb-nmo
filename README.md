# nmo

`nmo` is a cli tool to manage CouchDB clusters and nodes.

Requires [node](https://nodejs.org) >= 4.0.

## Installation

```
npm install -g nmo
```

## Contributing

It is written in ES6 and uses promises and generators for control flow.
